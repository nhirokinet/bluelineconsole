Blue Line Console は、キーボードで素早く必要なアプリやウェブ検索を開いたり、簡単な計算をしたりするためのランチャーです。

コマンドをキーボードから入力することで、快適に操作できます。パッケージ名（例：net.nhiroki.bluelineconsole）の一部でも呼び出せるため、たいていの場合はアルファベットを 2-3 文字入力すれば望むアプリが上に表示されます。アプリを並べる方式のランチャーと異なり、アプリの順序を設定する必要はありません。

物理キーのない端末でも、 Android デフォルトのアシストアプリに設定することにより、ホームボタン長押しで起動することができます。また、通知バーから呼び出し、ソフトウェアキーボードを用いることで、いつも同じようにアプリケーションを起動したり、ウェブ検索をしたりできます。（通知バーに表示する場合は config コマンドから設定する必要があります）

次の一覧のいずれかで検索できます。

- アプリケーション名（例： Blue Line Console）の一部
- パッケージ名（例： net.nhiroki.bluelineconsole）の一部
- URL
- 計算式（例： 2+3*5, 1inch in cm, 1m+1inch, 1m+1inch in cm）
- 下記のコマンド一覧のいずれか（例： help）

利用可能なコマンド:

- help
- config
- date
- bing QUERY
- duckduckgo QUERY
- google QUERY
- wikipedia QUERY
- yahoo QUERY
- ping HOST
- ping6 HOST
